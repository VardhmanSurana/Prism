**Key design decisions:**
- Animated WebP via `PIL.Image.save(save_all=True)` — Pillow already supports this, no new deps
- Seek to 10% of duration for representative frame (avoids black first frames)
- Cap at 400px width, 2s duration, ~12fps to keep file small
- If ffmpeg/ffprobe not installed, log warning and skip video thumbnails (graceful degradation)
- Content hash computation identical to image path for dedup consistency

```typescript
export function formatDuration(seconds: number): string {
  const h = Math.floor(seconds / 3600);
  const m = Math.floor((seconds % 3600) / 60);
  const s = Math.floor(seconds % 60);
  if (h > 0) return `${h}:${String(m).padStart(2, '0')}:${String(s).padStart(2, '0')}`;
  return `${m}:${String(s).padStart(2, '0')}`;
}
```
