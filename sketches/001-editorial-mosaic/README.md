## Variant: Editorial Mosaic

### Design stance
Magazine-inspired editorial layout — asymmetric grids, bold serif typography, and editorial dividers that create rhythm between sections. Think: Apple Photos meets Flipboard.

### Key choices
- **Layout:** Single-column with asymmetric hero grid (1.6fr + 1fr + 1fr)
- **Typography:** DM Serif Display for headings, Inter for body, JetBrains Mono for data
- **Color:** Deep OLED black (#050505) with subtle accent (#5e6ad2)
- **Interaction:** Scroll-reveal animations, hover lifts, image zoom
- **Dividers:** Icon-centered editorial dividers between sections (📅, 📊, ✨, 🌿, 📍)

### Trade-offs
- **Strong at:** Visual hierarchy, section separation, scan-ability
- **Weak at:** Dense information packing, at-a-glance stats
- **Best for:** Users who enjoy browsing and discovery — a "magazine" experience

### Differences from current
- Hero grid is asymmetric (featured story takes 2 rows) instead of equal cards
- Editorial dividers between every section create visual breathing room
- Stats are promoted to a 4-column metric grid instead of being nested in cards
- Timeline is horizontal cards instead of vertical timeline
- Seasonal uses portrait-aspect cards instead of squares
